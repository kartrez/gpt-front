Шесть шагов от установленного расширения до первого выполненного запроса Клёпы.

Перед стартом установите расширение — инструкция по установке .
1

Откройте Клёпу

Нажмите на иконку Клёпы в боковой панели VS Code (вертикальная панель сбоку окна), чтобы открыть интерфейс чата. Если иконки нет — убедитесь, что расширение установлено и включено.

Совет
Иконка Клёпы отображается в левой панели VS Code рядом с другими активными расширениями.
2

Авторизация через Telegram

Нажмите на иконку профиля в правом верхнем углу интерфейса Клёпы и выберите авторизацию через Telegram.

После авторизации нажмите кнопку возврата в VS Code, чтобы вернуться к интерфейсу Клёпы.

Важно
Авторизация требуется только один раз. После этого можно использовать Клёпу без повторного входа.
3

Введите задачу

Нажмите кнопку «Новая задача» и опишите своими словами, что нужно сделать. Никаких специальных команд — просто обычный русский язык.

«Создай файл hello.txt с содержимым «Hello, world!»»
«Напиши Python-функцию, которая складывает два числа»
«Сделай HTML-страницу с заголовком «Тест Клёпы»»
4

Отправьте задачу

Нажмите Enter или иконку отправки справа от поля ввода.

5

Подтверждение действий

Клёпа разбирает запрос и предлагает конкретные шаги. Возможные действия:

Чтение файлов
Показывает содержимое файлов, к которым нужен доступ
Запись в файлы
Diff с изменениями: зелёным — добавлено, красным — удалено
Выполнение команд
Показывает точную команду перед запуском
Работа с браузером
Описывает действия — клик, ввод текста и т.д.
Уточняющие вопросы
Запрашивает уточнения, если их не хватает
Режимы работы
В режиме Code запись включена по умолчанию. В режимах Architect и Ask Клёпа не будет писать код. Уровень автономии настраивается.
6

Итерации

Клёпа работает итеративно — после каждого действия ждёт вашей обратной связи. Продолжайте цикл проверки и подтверждения, пока задача не будет готова.

Преимущество
Итеративный подход позволяет ИИ уточнять и улучшать свою работу на основе ваших отзывов.
