Что выбрать

Klepa — если вы в РФ или РБ
  • Нет зарубежной карты — оплата МИР, БЕЛКАРТ, Visa и Mastercard банков РБ
  • Не хотите менять привычный VS Code на другой редактор
  • Нужны бесплатные модели для старта и оплата по факту вместо подписки
  • Важна поддержка и общение на русском языке
  • Хотите генерировать изображения прямо в редакторе
Cursor — если ограничений нет
  • Есть зарубежная банковская карта для оплаты подписки
  • Готовы перейти на отдельный редактор (форк VS Code)
  • Нужен фирменный Tab-автокомплит Cursor
  • Не смущают возможные блокировки аккаунтов из РФ и РБ
